# Hidden Tunes Mobile Freeze

The mobile app was verified on a physical device and frozen on 14 July 2026.

Stable tag:

`mobile-stable-2026-07-14`

Mobile is now in maintenance mode.

Allowed changes:

- Critical crash fixes.
- Security fixes.
- Store compliance fixes.
- Backend contract fixes required to preserve existing behaviour.
- Severe playback or navigation regressions.

Not allowed without explicit approval:

- New features.
- Redesigns.
- Navigation rewrites.
- Playback engine changes.
- Dependency upgrades.
- Large refactors.
- Experimental performance work.
- Desktop-specific code.
